# 労災用語 弾幕避けゲーム

労働問題に関する用語が弾幕として飛んでくる、マウス操作の弾幕避けゲームです。

## 起動手順

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## 操作方法

- **マウスを動かす**: 自機（青い丸）がマウスカーソルを追いかけて移動します
- 自機は即座ではなく、なめらかに遅れて追従します

## ゲームルール

1. 労災用語（「過労死」「パワハラ」など）が全方向から弾幕として飛んでくる
2. 弾に 1 発でも当たったらゲームオーバー
3. 生存時間（秒）がスコアとして表示される
4. 時間が経過するほど弾速・出現頻度が増し、難易度が上がる
5. ゲームオーバー後はリスタートボタンで再挑戦できる
